Project Lesuitval.info
Goal:
Hyperion Lyceum publishes changes in the school timetable per student on their website. Students have to check for cancelations of lessons by visiting this website. The goal of this project is to push this info to their phones automatically.
Solution:
Hyperion students register email and student number on my website. My server checks for timetable changes regularly and send pushbullets to their phones whenever a cancelation of lessons is detected. Pushbullet is a push application available for most phones.
Project details:

  • Server runs on digitalocean VPS with ubuntu 14.10 x32 installed.
  • I use the Django Framework with Gunicorn under Supervisorctl.
  • Static files by Nginx.
  • SQLite3 for the database.
  • Cron job runs a bash file that runs a Python script
  • Pushbullet communication by Python library Pushbullet.py
  • Python script:
    1. Detects time table changes with Regular expressions on individualized website:
      http://gepro.nl/roosters/rooster.php?wijzigingen=1&leerling=1102053&type=Leerlingrooster&afdeling=schooljaar2014-2015_OVERIG&school=1814
    2. where "leerlingnummer" 1102053 varies per student.
  • Input validation:
    1. Only valid email addresses
    2. "leerlingnummer" must be a number
    3. Unsubscribing requires existing email and leerlingnumber in database
  • Domain registration Lesuitval.info
  • I use Bootstrap3 for css webdesign
  • Code management by Github versioning
